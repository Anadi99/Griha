# PlanAura

PlanAura is a mobile app for designing 2D floor plans with smart Vastu-based suggestions and cost estimation.

## Current Features

* Canvas-based room design (drag, resize)
* Room types (bedroom, kitchen, etc.)
* Basic UI for designing layouts
